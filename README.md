# Interactive Coffee Sales Dashboard

This is a recruiter-ready portfolio website generated from the uploaded Excel workbook `coffeeOrdersData.xlsx`.

## What is included

- Interactive filters: year, country, coffee type, roast type, package size, loyalty card, and search.
- KPI cards: total sales, profit, order lines, units sold, average order line, and top country.
- Charts: monthly trend, coffee type revenue, country revenue, top customers, roast mix, and size performance.
- Data explorer with sortable columns and filtered CSV download.
- Original Excel source file in `data/coffeeOrdersData.xlsx`.

## Dataset summary

- Rows: 1000
- Date range: 2019-01-02 to 2022-08-19
- Total sales: $45,134.25
- Countries: Ireland, United Kingdom, United States
- Coffee types: Arabica, Excelsa, Liberica, Robusta

## How to use

Open `index.html` in a browser. For best chart rendering, keep an internet connection so Chart.js can load from CDN.
